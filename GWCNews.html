Hour of Code

The Hour of Code is a one-hour introduction to computer science sponsored by Computer Science Education Week and Code.org. Its goal is to remove some of the mystery around programming and show participants that anyone can code. It has become a global movement and has reached tens of millions of students in 180+ countries.

The Notre Dame Girls Who Code Club has taken ownership for Notre Dame's Hour Of Code for the last three years. We have developed online games and classroom activities to introduce ND freshman and sophomores to programming concepts. We designed coding exercises for different levels, from beginner to intermediate programmers, so anyone can participate. (If you're already an expert, you should volunteer to help out!)

To learn more about Hour Of Code, check out this website!
