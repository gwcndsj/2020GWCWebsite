Coding Resources

Wanna learn how to code? Here's a bunch of great sites that offer coding tutorials!

Links to introductory videos and example projects.

Scratch

Scratch is a free graphical “drag and drop” environment that allows you to build games, animations, etc using programming “building blocks”.

Link to an introductory tutorial, which is also a good resource for learning about programming in general.

Python

Python is a general purpose programming lnaguage that's very popular. It can be used to program web pages, create games, or anything else.

Link to some tutorials. Khan Academy is a great way to ease into programming

Javascript

The Javascript language is very useful website development. The tutorial on Kahn Academy’s website is a series of instructional videos followed by labs that can be done in the same browser session. The instructional videos show the process of programming, making mistakes and fixing them.
